Problemas en la fase de desarrollo

Los más comunes (experiencia personal)

  • Diferencia de ambientes (Linux, OS X, Windows)
  • Diferencias entre versiones del "mismo" ambiente o bibliotecas
  • Diferentes capacidades técnicas (Desarrolladores, Administradores de Servidores, Diseñadores, QA, Management)

Soluciones existentes

  • Servidor de desarrollo
  • Máquinas virtuales
  • Vagrant
  • Docker

WP-CLI

Un conjunto de herramientas de línea de comandos para administrar instalaciones de WordPress.

Descargar e Instalar

Ver Dockerfile, build.sh y ejemplo.sh en la carpeta de ejemplo 3.

WP-CLI

  • Installar plugins: wp plugin install kint-debugger
  • Desinstalar plugins: wp plugin uninstall kint-debugger
  • Activar plugins: wp plugin activate kint-debugger
  • Desactivar plugins: wp plugin deactivate kint-debugger
  • Installar temas: wp theme install _tk
  • Desinstalar temas: wp plugin uninstall _tk
  • Activar temas: wp plugin activate _tk
  • Desactivar temas: wp plugin deactivate _tk
  • Limpiar cache: wp cache flush
